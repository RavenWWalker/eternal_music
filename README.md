# 🔔 Музыкальная шкатулка

Бесконечная процедурная мелодия — каждое прослушивание уникально.
Колокольчики и тёплый пэд, сгенерированные алгоритмически.

**🎧 [Послушать в браузере](https://ВАШ_НИК.github.io/music-box/)**

## Что это

Программа в реальном времени генерирует звук с нуля — никаких сэмплов, никаких готовых аудиофайлов. Колокольчики строятся из синусоид с негармоническими обертонами (как у настоящих колоколов), пэд — из расстроенных осцилляторов с медленным фильтром. Аккордовые прогрессии и арпеджио выбираются случайно.

Доступно в двух вариантах: веб-версия для браузера и Python-приложение.

## Веб-версия

Просто откройте сайт по ссылке выше — никакой установки не нужно. Работает в любом современном браузере (Chrome, Firefox, Safari, Edge).

## Python-версия

### Требования
- Python 3.8 или новее
- pip

### Установка

```bash
git clone https://github.com/ВАШ_НИК/music-box.git
cd music-box
pip install -r requirements.txt
```

### Запуск

```bash
python music_box.py
```

Чтобы остановить — нажмите Enter в терминале.

### Регулировка громкости

Откройте `music_box.py` и найдите:
- В методе `play_bell` — последнее число в строке `vol = velocity * ... * 0.55` отвечает за громкость колокольчиков
- В методе `play_pad` — `s.set_volume(0.9)` отвечает за громкость пэда

## Как это работает

Звук — это колебания воздуха. Динамик в колонке или наушниках движется вперёд-назад, толкая воздух, и так возникает звук. Программа 44100 раз в секунду решает, где сейчас должна быть мембрана. Получается длинная последовательность чисел — это и есть цифровой звук.

Чтобы звук стал колокольчиком, программа складывает несколько синусоид с особыми соотношениями частот (1.0, 2.0, 2.76, 5.40, 8.93 — именно такие пропорции дают характерный «дзинь» настоящего колокола). Каждая синусоида затухает со своей скоростью — это создаёт ощущение живого инструмента.

Пэд работает похоже, но осцилляторы слегка расстроены друг относительно друга, и сверху накладывается медленный фильтр, который «открывается» и «закрывается». Это даёт ощущение дышащего звукового полотна.
